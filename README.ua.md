### 📘 Опис:
Це проста програма для підрахунку та аналізу переглянутих відео (наприклад, у TikTok, Reels, Shorts). 
Загальна мета: передивитися 100 або більше відео та розподілити їх за категоріями:
  - 🟢 Добре / Корисне
  - 🟡 Нейтральне
  - 🔴 Погане / Деградуюче

Цей додаток допомагає зробити аналіз споживання контенту простішим і наочнішим.


### 🏗️ Архітектура проекту:

Дотримуючись принципу розділення відповідальності (Separation of concerns), структура проекту суворо поділена на вихідний код, дані користувача та файли локалізації:

```bash
OneHundredClips/

├── locales/                  # 🌐 Файли локалізації (i18n)
├── user_data/                # 📂 Дані, згенеровані користувачем
│   ├── save/                 # Історія щоденних звітів
│   ├── settings.json         # Налаштування та власні критерії
│   └── data.json             # Поточна сесія підрахунку
├── src/                      # 💻 Вихідний код програми
│   ├── config.py             # Глобальні налаштування та завантаження мов
│   ├── storage.py            # Логіка збереження даних
│   ├── app_logic.py          # Головна математика та логіка програми
│   └── gui/                  # 🎨 Графічний інтерфейс (GUI)
│       ├── main_window.py    # Налаштування головного вікна та сітки
│       ├── windows/          # Окремі спливаючі вікна (Історія, Редактори тощо)
│       └── components/       # Багаторазові UI-компоненти
├── assets/                   # 📸 Скріншоти для README
├── README.ua.md              # Документація проекту
└── main.py                   # 🚀 Точка входу в програму
```
