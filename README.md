# Analysis 100 TikTok 

---

### Platform \ Пратформа:

**PC - Windows**

### Tech Stack \ Технології: 

- **Python 3.12**
- **Tkinter (GUI \ Графічний інтерфейс)**
- **JSON (Data Persistence \ Збереження даних)**

### Description \ Опис:

**Це проста програма для підрахунку оцінок пререглянутих відео за критеріями:**
**добре, нейтральне, погане\негативне.**

**Загальною метою є: передивитися 100 TikTok-відео і розділити їх по критеріям,**
**ця програма допоможе зробити аналіз простішим, за результатами можна підвести** 
**підсумок, чи було корисним або приємний переглядати ці 100 відео**

### Future improvements \ Майбутне вдосконалення:

1. Додання можливості вибору тексту/емоджи при натискані кнопки
2. Детальніший звіт-результат з результату **1**
3. Створення власного тексту/емоджи до кнопки в самому інтерфейсі (одноразово)
4. Редагування тексту критерій(додати власний критерій) в самому інтерфейсі (с зебеженням)
5. Покрашення чи переробка інтерфецсу
6. Інше...
